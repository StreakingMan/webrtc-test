# WebRTC 1v1 方块移动演示

这是一个使用 PeerJS 实现的简单 WebRTC 演示项目，展示了如何实现两个玩家之间的实时方块移动同步。

## 功能特点

- 使用 PeerJS 实现 P2P 连接
- 实时同步双方方块位置
- 使用方向键控制本地方块移动

## 如何使用

1. 直接在浏览器中打开 `index.html` 文件
2. 等待获取本地ID
3. 将本地ID发送给对方
4. 输入对方ID并点击"连接"按钮
5. 连接成功后，使用方向键控制红色方块移动
6. 蓝色方块代表对方位置

## 控制说明

- ↑: 向上移动
- ↓: 向下移动
- ←: 向左移动
- →: 向右移动

## 技术说明

- 使用 PeerJS 免费信令服务器
- 原生 JavaScript 实现
- Canvas 绘制游戏画面
